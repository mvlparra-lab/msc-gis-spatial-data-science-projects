# GeoJSON Processing and Spatial Queries

Python project developed to process GeoJSON spatial data and perform spatial query operations through a Jupyter Notebook workflow.

## Objectives

The workflow combines several Python geospatial libraries to read, transform, analyse and export vector spatial data in GeoJSON format.

## Main Tasks

- Reading GeoJSON layers with Fiona
- Geometry and attribute exploration
- Coordinate transformation using PyProj
- Spatial operations with Shapely
- Point in polygon analysis
- Spatial feature counting by district
- New GeoJSON output generation
- GeoDataFrame analysis with GeoPandas
- CRS inspection and spatial data exploration

## Technologies

- Python
- Jupyter Notebook
- fiona
- shapely
- pyproj
- geopandas
