# Exploratory Spatial Data Analysis of Barcelona Neighborhoods

Spatial analysis project developed to explore socioeconomic inequalities and spatial clustering patterns across the neighborhoods of Barcelona using ESDA techniques.

## Objectives

The workflow combines exploratory statistics and spatial autocorrelation analysis to identify relationships between unemployment, income, housing, economic activity and urban vulnerability indicators.

## Study Area

- Barcelona, Spain
- 73 official neighborhoods

## Main Tasks

- Spatial and socioeconomic data integration
- Attribute and spatial joins
- Univariate exploratory analysis
- Bivariate exploratory analysis
- Multivariate spatial analysis
- Spatial weights matrix creation
- Spatial autocorrelation analysis
- Local Moran’s I (LISA) analysis
- Local Geary analysis
- Multivariate spatial clustering
- Vulnerability pattern detection

## Technologies

- GeoDa
- QGIS
- Microsoft Excel
