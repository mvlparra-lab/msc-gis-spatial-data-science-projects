# Exploratory Data Analysis in R

R project developed to perform exploratory analysis of socioeconomic and health related data from Massachusetts counties.

## Objectives

The workflow combines data cleaning, statistical exploration, outlier detection and visualization techniques to analyse relationships between income, health, population and life expectancy.

## Main Tasks

- Dataset import and initial exploration
- Column name standardization
- Missing value detection and treatment
- Derived analytical variable creation
- Outlier detection and visualization
- Descriptive statistical analysis
- Socioeconomic and health relationship analysis
- Publication ready visualization generation
- Export of statistical summaries and reports
