# Data Cleaning and Preparation in R

R project developed to clean, standardize and prepare a structured dataset for exploratory analysis and future statistical workflows.

## Objectives

The workflow transforms a raw dataset containing missing values, duplicated records, typographical inconsistencies and formatting issues into a clean and analysis ready dataset.

## Main Tasks

- Dataset import and initial exploration
- Column name standardization
- Missing value detection and management
- Typographical error correction
- Text normalization and capitalization
- Duplicate record detection and removal
- Structured information extraction
- BMI calculation and classification
- Export of cleaned datasets

## Technologies

- R
- RStudio
- tidyverse
- janitor
- lubridate
- stringr
- skimr
