# Visual Impact Analysis of Wind Turbines

Spatial analysis project developed in R to evaluate the visual impact generated by wind turbines in the comarca of La Segarra (Lleida, Spain).

## Objectives

The project combines raster and vector geoprocessing techniques to analyse terrain visibility, identify areas under maximum visual impact, and detect cadastral parcels affected by wind turbine visibility.

## Main Tasks

- DEM preprocessing and terrain analysis
- Hillshade generation
- Elevation threshold classification (>750 m)
- Wind turbine location selection
- Viewshed computation
- Cumulative visual impact analysis
- Municipal impact aggregation
- Land cover impact assessment
- Cadastral parcel retrieval using WFS (Catastro)
- Spatial intersection between parcels and maximum impact zones
- Export of cartographic and tabular outputs

## Technologies

- R
- sf
- terra
- dplyr
- ggplot2
- CatastRo
