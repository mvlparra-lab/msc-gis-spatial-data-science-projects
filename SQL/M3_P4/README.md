# Spatial SQL Queries and Meteorological Analysis with PostGIS

SQL and PostgreSQL/PostGIS project developed to import spatial datasets and perform advanced spatial and meteorological queries.

## Objectives

The workflow focuses on spatial database management, geospatial data import and execution of advanced SQL and PostGIS queries for meteorological and air quality analysis.

## Main Tasks

- SHP import using shp2pgsql
- PostgreSQL and PostGIS database configuration
- Spatial table validation
- Primary key definition
- Spatial joins using PostGIS functions
- Meteorological data analysis
- Air quality query analysis
- Daily precipitation aggregation
- Spatial precipitation analysis by municipality and comarca
- Advanced SQL queries using CTEs and spatial functions

## Technologies

- SQL
- PostgreSQL
- PostGIS
- pgAdmin
- shp2pgsql
