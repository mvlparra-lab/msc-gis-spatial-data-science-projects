# Physical Design of a Spatial Database

SQL and PostgreSQL/PostGIS project developed to implement the physical design of a spatial database from a logical relational model.

## Objectives

The workflow focuses on converting a logical database design into a functional spatial database using PostgreSQL, PostGIS and pgAdmin.

## Main Tasks

- PostgreSQL database creation
- PostGIS extension activation
- SHP import using shp2pgsql
- Spatial table verification with SELECT queries
- Table modification using ALTER TABLE
- Primary key definition
- Table creation using CREATE TABLE
- Foreign key relationship creation
- Manual data insertion using INSERT INTO
- Bulk data import using COPY
- Spatial database validation

## Technologies

- SQL
- PostgreSQL
- PostGIS
- pgAdmin
- shp2pgsql
